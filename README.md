# اذكار (Azkar) - منصتك الإسلامية الشاملة

**اذكار** هو تطبيق ويب إسلامي شامل ومفتوح المصدر، يهدف إلى توفير الأدوات والوسائل اللازمة لكل مسلم في حياته اليومية بصورة ميسرة وجذابة. يتميز الموقع بتصميم عصري (باللون الأزرق المريح) وسرعة في الأداء، مع واجهة مستخدم متجاوبة تعمل بكفاءة على جميع الأجهزة (الهواتف، الأجهزة اللوحية، والحواسيب).

---

## 🌟 المميزات الرئيسية

- **📖 القرآن الكريم:** تصفح وقراءة القرآن الكريم كاملاً مع تلاوة صوتية بصوت القارئ مشاري راشد العفاسي، مع إمكانية البحث عن السور والتحكم في حجم الخط.
- **🙏 الأذكار:** قسم شامل للأذكار اليومية (أذكار الصباح، المساء، النوم، الاستيقاظ، الصلاة، وغيرها) مع عداد إلكتروني لكل ذكر وإمكانية إزالة التشكيل لتسهيل القراءة.
- **🕒 مواقيت الصلاة:** عرض دقيق لمواقيت الصلاة بناءً على موقعك الجغرافي (Geolocation) مع عداد تنازلي للصلاة القادمة.
- **📿 السبحة الإلكترونية:** عداد تسبيح رقمي بسيط مع إمكانية حفظ سجل التسبيحات واختيار جمل الذكر.
- **📚 الحديث الشريف:** تصفح الأحاديث النبوية من الكتب الستة (البخاري، مسلم، الترمذي، النسائي، أبو داود، ابن ماجه).
- **⭐ أسماء الله الحسنى:** قائمة كاملة بأسماء الله الحسنى الـ 99 مع معانيها بطريقة عرض جميلة.
- **📅 التاريخ الهجري:** عرض التاريخ الهجري والميلادي المحدث تلقائياً.

---

## 🛠️ التقنيات المستخدمة

تم بناء المشروع باستخدام تقنيات الويب الأساسية لضمان أقصى درجات التوافق والسرعة:

- **HTML5:** للهيكلة الأساسية.
- **CSS3:** للتصميم الجمالي والتجاوب (Responsive Design) باستخدام CSS Grid و Flexbox.
- **JavaScript (Vanilla JS):** لإدارة المنطق البرمجي والتفاعل مع المستخدم دون الحاجة لمكتبات ضخمة.
- **Font Awesome:** للأيقونات.
- **Google Fonts:** لاستخدام خطوط "Amiri" و "Cairo".
- **External APIs:**
  - [Aladhan API](https://aladhan.com/prayer-times-api) لمواقيت الصلاة.
  - [Al-Quran Cloud API](https://alquran.cloud/api) لبيانات القرآن الكريم.
  - [Hadith API](https://hadithapi.com/) لبيانات الأحاديث النبوية.
